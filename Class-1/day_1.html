Competitive Programming Class Routine

Week 1: Introduction to Competitive Programming and C Basics

Day 1: Introduction to Competitive Programming and C Basics

Time: 45 minutes

  1. Introduction to the course and its objectives.
  2. Explanation of what competitive programming is and its significance.
  3. Overview of the C programming language and its importance in competitive programming.

Day 2: Variables, Data Types, and Basic I/O in C

Time: 45 minutes

  1. Detailed explanation of variables and their types in C.
  2. Introduction to data types: int, float, double, char, etc.
  3. In-depth discussion of input and output functions using scanf and printf.

Week 2: Control Structures and Problem-Solving Framework

Day 3: Control Structures in C

Time: 45 minutes

  1. Detailed explanation of conditional statements: if, else if, else.
  2. Introduction to loops: for, while, do-while.

Day 4: Problem-Solving Framework and Basic Algorithmic Concepts

Time: 45 minutes

  1. Step-by-step problem-solving framework: read, example, plan, code, test, optimize.
  2. Introduction to basic algorithmic concepts: brute force, greedy algorithms.
